# Planning ZeShoes & LGL

Application calendrier simple pour planning ZeShoes et LGL.

## Fonctionnalités
- Navigation mois/année (flèches, sélecteur manuel, swipe tactile)
- Ajout/édition noms par jour (modal)
- Persistance localStorage (anti-perte)
- Impression planning
- Responsive + support mobile
- Raccourcis clavier (Escape)

## Utilisation
Ouvrir `index.html` dans navigateur. Données auto-sauvegardées.
